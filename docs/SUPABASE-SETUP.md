# ربط فزعتك بقاعدة بيانات Supabase

هذا الدليل ينقل تخزين **فزعتك** من المحلي (localStorage) إلى قاعدة بيانات
Supabase حقيقية. بقية صفحات الموقع (المراكز عبر الخريطة الحيّة، الحملات عبر
المصادر الحيّة) لا تحتاج قاعدة بيانات.

> مهم: هذا يجعل التخزين **حقيقيًا**، لكنه لا يجعل المنصة **رسمية**. تبقى تنبيهات
> «نموذج غير رسمي / لا يستخدم لطلبات الدم الحقيقية / تواصل مع المستشفى» قائمة حتى
> استيفاء متطلبات الإطلاق الفعلي في `docs/FUTURE-OFFICIAL-INTEGRATION.md`
> (تنسيق مع المستشفيات، تحقق هوية، مصادقة، مراجعة قانونية/خصوصية).

## 1) أنشئ المشروع وخذ المفاتيح
من Supabase: `Project URL` و`anon public key`.

## 2) شغّل الهجرة
افتح **SQL Editor** في Supabase والصق محتوى:
`supabase/migrations/0001_faz3tak.sql` ثم نفّذه.

التصميم الأمني:
- RLS مفعّل ولا وصول مباشر للجداول من المتصفح.
- القراءة العامة عبر العرض `faz3_requests_public` الذي **يخفي رقم الهاتف** ويعرض
  الاسم حسب خيار الخصوصية فقط.
- الإنشاء والتحديث عبر دوال RPC تتحقق من **رمز الإدارة** (مخزَّن كتجزئة) وتطبّق
  القواعد (عدم تجاوز الهدف، الإغلاق التلقائي عند الاكتمال) خادم-جانبيًا.
- إجراءات الإشراف (إخفاء/إزالة) **لا تُمنح للمفتاح العام**؛ تُنفَّذ عبر
  `service_role` من بيئة آمنة فقط.

## 3) أضف متغيّرات البيئة
محليًا في `.env.local`:
```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
```
للنشر على GitHub Pages: أضِفهما إلى **Settings → Secrets and variables → Actions**
(كـ Variables أو Secrets)، ومرّرهما في خطوة البناء داخل `.github/workflows/deploy.yml`:
```yaml
      - name: Build static site
        run: npm run build
        env:
          NEXT_PUBLIC_SUPABASE_URL: ${{ secrets.NEXT_PUBLIC_SUPABASE_URL }}
          NEXT_PUBLIC_SUPABASE_ANON_KEY: ${{ secrets.NEXT_PUBLIC_SUPABASE_ANON_KEY }}
```
(لأن متغيّرات `NEXT_PUBLIC_*` تُدمج وقت البناء في التصدير الثابت.)

## 4) السلوك
- بوجود المفتاحين: يُفعّل `isSupabaseEnabled` ويُستخدم Supabase لفزعتك.
- بدونهما: يبقى التطبيق على التخزين المحلي دون أي كسر.

## تحذيرات أمان
- `anon key` عام؛ الأمان كله على RLS والدوال. راجِع السياسات قبل أي استخدام حقيقي.
- لا تضع `service_role key` في كود المتصفح أو في بناء عام.
- لا تُخزّن بيانات طبية أو رقمًا وطنيًا؛ رقم الهاتف خاص ولا يظهر في العرض العام.
