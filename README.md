# وصّل دم | Wasl Dam

نموذج تجريبي (Web Prototype) لمنصة دعم رقمية تساعد الناس على إيجاد مراكز التبرع
بالدم الرسمية، متابعة الحملات المعتمدة، وتسجيل نيّتهم للتبرع.

> **نموذج تجريبي تابع لمبادرة همّة الشبابية (NYIJO).**
> هذا المشروع لأغراض العرض والاختبار فقط، **وغير مرتبط بأي بنك دم رسمي أو وزارة
> الصحة**، ولا يمثّل نظامًا رسميًا. جميع البيانات المعروضة **بيانات تجريبية**.

---

## ما الذي تقدّمه المنصة

1. البحث عن مراكز تبرع رسمية (بيانات تجريبية) حسب الاسم والمحافظة.
2. استعراض حملات التبرع التجريبية مع تفاصيلها وتسجيل نية التبرع.
3. عرض حالة الطلب على الدم حسب الفصيلة والمحافظة (متوفر / بحاجة دعم / احتياج عاجل).
4. تسجيل الاستعداد للتبرع (يُحفظ محليًا في المتصفح فقط).
5. فحص توعوي مبدئي للأهلية (بدون تشخيص أو تخزين).
6. أسئلة شائعة وإرشادات توعوية غير طبية.

## ما الذي لا تقدّمه المنصة

- ليست بنك دم ولا تخزّن دمًا.
- لا تربط المتبرعين بالمرضى.
- لا تجمع الرقم الوطني أو بيانات طبية أو سجلات مستشفيات.
- لا تعرض كميات مخزون فعلية ولا بيانات شخصية للمتبرعين.
- لا تقدّم تشخيصًا أو موافقة طبية. القرار النهائي للتبرع يعود لكادر بنك الدم الرسمي.

راجع [docs/SAFETY.md](docs/SAFETY.md) و
[docs/FUTURE-INTEGRATION.md](docs/FUTURE-INTEGRATION.md) للمزيد.

## التقنيات

- Next.js (App Router) + TypeScript
- Tailwind CSS (مع دعم الوضع الداكن و RTL)
- بيانات تجريبية محلية فقط (`/data`)
- `localStorage` للمتصفح لبعض الإجراءات التجريبية
- بدون خادم خلفي أو قاعدة بيانات أو مصادقة حقيقية أو أي واجهة برمجية خارجية

## التشغيل محليًا

```bash
npm install
npm run dev
```

ثم افتح: http://localhost:3000

## الأوامر

```bash
npm run dev        # تشغيل بيئة التطوير
npm run build      # بناء الإنتاج
npm run start      # تشغيل بناء الإنتاج
npm run lint       # فحص ESLint
npm run typecheck  # فحص أنواع TypeScript
```

## بنية المشروع

```
app/         # صفحات Next.js (App Router)
components/  # مكوّنات واجهة قابلة لإعادة الاستخدام
lib/         # أنواع وأدوات مساعدة وتخزين محلي
data/        # بيانات تجريبية (مراكز، حملات، حالة الطلب)
public/      # ملفات ثابتة (الشعار)
docs/        # وثائق السلامة والتكامل المستقبلي
```

## ملاحظة حول البيانات التجريبية

جميع المراكز والحملات وحالات الطلب والإحصاءات **بيانات خيالية لأغراض العرض فقط**،
وتظهر عليها بطاقات "بيانات تجريبية". لا تُستخدم أي بيانات مخزون حقيقية أو جداول
مستشفيات أو تفاصيل رسمية.

## لوحة العرض التجريبية

تتوفر لوحة تجريبية على `/demo-admin` (كلمة مرور تجريبية: `demo1234`) لتعديل
البيانات محليًا في المتصفح فقط. **ليست لوحة إنتاج حقيقية وغير مرتبطة بوزارة الصحة.**
